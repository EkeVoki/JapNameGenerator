# 🎌 Japanese Name Generator

**A simple name generator for your characters, nicknames, or D&D campaigns.**

## 📖 About the project

This generator was written in about **30 minutes** for my personal use.  
But since it’s now public, it’s no longer just for me — it’s for anyone who needs a quick, random Japanese-style name.

**Use it for:**

- Nicknames or usernames
- Fanfiction characters
- D&D / RPG sessions
- Or just for fun (heh~)

## 🌟 What's new in version 1.1

- 🌐 **Language selection** – now you can choose between English and Russian
- 💾 **Save to file** – all generated names can be saved to a `.txt` file
- 🧹 **Improved error handling** – default language if input is incorrect
- 📁 **Auto‑save folder** – files are saved in the same folder as the script
